## Full Description
A modern TypeScript boilerplate implementing Clean Architecture principles with hexagonal architecture. Features Express.js, PostgreSQL with Drizzle ORM, JWT authentication, comprehensive testing with Jest, background job processing with BullMQ, and auto-generated OpenAPI documentation. Perfect for building scalable, maintainable backend applications.
